# Data Generation Scripts

## Purpose

This directory contains scripts for generating mock data, test data, and visual assets for the project.

## Contents

- **generate_mock_carbon_data.py**: Generate mock carbon emission data for testing
- **generate_carbon_emission_tables.py**: Generate carbon emission calculation tables
- **generate_carbon_dashboard_images.py**: Generate dashboard screenshot images
- **generate_epa_document_images.py**: Generate EPA document images
- **generate_pwa_icons.py**: Generate PWA icon assets

## Usage

Run scripts directly from the project root:

```bash
python scripts/data_generation/generate_mock_carbon_data.py
```

## Related Directories

- `data/databases/` - Generated database files
- `assets/images/` - Generated image assets
- `static/icons/` - Generated PWA icons
