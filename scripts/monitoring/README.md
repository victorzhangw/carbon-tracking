# Monitoring Scripts

## Purpose

This directory contains scripts for monitoring system status, debugging, and performance tracking.

## Contents

- **monitor_deployment.py**: Monitor deployment status and health
- **show_all_workers_stats.py**: Display statistics for all social workers
- **debug_voice_models.py**: Debug voice model issues

## Usage

Run monitoring scripts from the project root:

```bash
python scripts/monitoring/monitor_deployment.py
```

## Related Directories

- `data/logs/` - Log files for monitoring
- `services/` - Services being monitored
- `modules/` - Modules being monitored
