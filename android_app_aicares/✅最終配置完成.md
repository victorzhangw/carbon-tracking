## 🚀 立即開始

### 方法 1：Android Studio（推薦）

1. **開啟專案**

   ```
   File → Open → 選擇 android_app_aicares 資料夾
   ```

2. **等待同步**

   - 首次同步需要 2-5 分鐘
   - 會自動下載 Gradle 和依賴套件
   - 底部狀態列顯示進度

3. **建置 APK**

   ```
   Build → Build Bundle(s) / APK(s) → Build APK(s)
   ```

4. **安裝到手機**
   - 連接手機（開啟 USB 偵錯）
   - 點擊 ▶️ Run 按鈕

### 方法 2：命令列

```bash
# 進入專案目錄
cd android_app_aicares

# 清理專案
gradlew clean

# 建置 Debug APK
gradlew assembleDebug

# 建置 Release APK（需要簽名）
gradlew assembleRelease
```

### 方法 3：使用批次檔

直接執行：

```
測試建置.bat
```

## 📱 測試步驟

### 1. 基本功能測試

```
1. 啟動 APP
   → 應該看到 Splash 畫面（紫色漸層）

2. 進入主畫面
   → 應該載入伺服器首頁

3. 測試導航
   → 點擊各個功能連結
   → 返回鍵應該能回到上一頁

4. 測試重新整理
   → 下拉應該能重新載入頁面
```

### 2. 權限測試

```
1. 首次啟動
   → 應該請求麥克風權限

2. 授予權限
   → 語音功能應該能正常使用

3. 拒絕權限
   → 應該顯示提示訊息
   → 其他功能仍可使用
```

### 3. 網路測試

```
1. WiFi 連線
   → 確認手機和電腦在同一 WiFi
   → 應該能正常連接

2. 無網路
   → 應該顯示「無網路連線」對話框
   → 提供重試選項

3. 伺服器離線
   → 應該顯示錯誤頁面
   → 提供重新載入按鈕
```

## 🐛 如果遇到問題

### Gradle 同步失敗

```bash
# 清除快取
gradlew clean
gradlew --stop

# 刪除 .gradle 資料夾
# 重新開啟 Android Studio
```

### 建置失敗

```bash
# 查看詳細錯誤
gradlew assembleDebug --stacktrace --info
```

### 無法連接伺服器

1. 確認伺服器正在運行
2. 在瀏覽器測試：http://192.168.1.102:5000
3. 檢查防火牆設定
4. 確認手機和電腦在同一網路
