## 🔄 如何更換為自訂圖示

### 方法 1：使用 Android Studio（推薦）

1. **開啟 Image Asset 工具**

   ```
   右鍵點擊 app → New → Image Asset
   ```

2. **選擇圖示類型**

   - Launcher Icons (Adaptive and Legacy)

3. **上傳圖片**

   - 選擇你的圖示檔案（建議 512x512 PNG）
   - 調整大小和位置

4. **生成圖示**

   - 點擊 Next → Finish
   - 會自動生成所有尺寸的圖示

5. **更新 AndroidManifest.xml**
   ```xml
   android:icon="@mipmap/ic_launcher"
   android:roundIcon="@mipmap/ic_launcher_round"
   ```

### 方法 2：手動創建

需要創建以下尺寸的圖示：

```
mipmap-mdpi/ic_launcher.png (48x48)
mipmap-hdpi/ic_launcher.png (72x72)
mipmap-xhdpi/ic_launcher.png (96x96)
mipmap-xxhdpi/ic_launcher.png (144x144)
mipmap-xxxhdpi/ic_launcher.png (192x192)
```

### 方法 3：線上工具

使用線上圖示生成器：

- https://romannurik.github.io/AndroidAssetStudio/
- https://icon.kitchen/
- https://appicon.co/

上傳圖片後下載生成的資源包，解壓到 `res/` 目錄。

## 🎯 圖示設計建議

### 尺寸

- **原始圖片：** 512x512 或 1024x1024
- **格式：** PNG（透明背景）
- **內容區域：** 中心 80% 區域

### 風格

- **簡潔：** 避免過多細節
- **識別度：** 一眼就能認出
- **配色：** 與 APP 主題一致（紫色系）

### 主題建議

- 🤖 AI 機器人
- 💬 對話氣泡
- 🎤 麥克風
- 🔊 音波
- 💜 紫色漸層

## 🎨 進階：Adaptive Icon

Android 8.0+ 支援 Adaptive Icon（自適應圖示）：

**創建 mipmap-anydpi-v26/ic_launcher.xml：**

```xml
<?xml version="1.0" encoding="utf-8"?>
<adaptive-icon xmlns:android="http://schemas.android.com/apk/res/android">
    <background android:drawable="@drawable/ic_launcher_background"/>
    <foreground android:drawable="@drawable/ic_launcher_foreground"/>
</adaptive-icon>
```

這樣圖示可以適應不同的形狀（圓形、方形、圓角方形等）。
