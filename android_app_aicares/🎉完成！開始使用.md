## 📱 安裝到手機

### 方法 1：USB 連接（推薦）

1. 手機開啟「開發者選項」
2. 開啟「USB 偵錯」
3. 用 USB 線連接電腦
4. 在 Android Studio 點擊 ▶️ Run

### 方法 2：直接安裝 APK

1. 將 APK 傳到手機（透過 USB、藍牙、雲端等）
2. 在手機上找到 APK 檔案
3. 點擊安裝
4. 允許「安裝未知來源」
5. 完成安裝

## 🌐 連接伺服器

### 確認伺服器運行中

在電腦瀏覽器開啟：

```
http://192.168.1.102:5000
```

- 手機和電腦必須在同一個 WiFi 網路
- 檢查防火牆設定（可能需要允許 5000 埠）

### 測試連接

1. 開啟 APP
2. 應該會自動載入伺服器頁面
3. 如果無法連接，會顯示錯誤頁面

## 🐛 常見問題

### 問題 1：無法連接伺服器

**解決方法：**

1. 確認伺服器正在運行
2. 確認手機和電腦在同一 WiFi
3. 檢查 IP 地址是否正確
4. 暫時關閉防火牆測試

### 問題 2：APP 閃退

**解決方法：**

1. 檢查 Android 版本（需要 7.0+）
2. 清除 APP 資料後重試
3. 重新安裝 APP

### 問題 3：權限被拒絕

**解決方法：**

1. 設定 → 應用程式 → AI 客服系統
2. 權限 → 開啟所需權限
3. 重新啟動 APP

### 問題 4：圖示不好看

**解決方法：**

- 參考 `✅圖示已創建.md`
- 使用 Android Studio 的 Image Asset 工具
- 上傳自訂圖示
