# Image Assets

## Purpose

This directory contains all image files used in the project, including icons, screenshots, and other visual assets.

## Contents

- `icons/` - Application icons and UI icons
- `screenshots/` - System screenshots for documentation

## Usage

Image files in this directory are referenced by:

- Frontend templates (`templates/`)
- Static resources (`static/`)
- Documentation (`docs/`)

## File Organization

- Keep images organized by type in subdirectories
- Use descriptive filenames
- Optimize images for web use when applicable

## Related Directories

- `static/icons/` - Icons served to frontend (may contain symlinks)
- `docs/` - Documentation that references screenshots
