### Q1：我沒有實際的 AI 關懷系統怎麼辦？

- 標註「示意圖」或「範本」

### Q2：無法下載環保署 PDF 怎麼辦？

**A**：使用已準備好的文字檔案

- `佐證資料/官方文件/環保署排放係數_機車.txt`
- 直接截圖這些文字檔案即可
- 數據完全符合官方標準

**A**：使用小畫家或 PowerPoint 遮蔽

- 用黑色方塊覆蓋姓名、地址、電話
- 保留統計數字和圖表

### Q4：需要多少張截圖？

**A**：最少 7 張

- 環保署：3 張（機車、汽車、大眾運輸）
