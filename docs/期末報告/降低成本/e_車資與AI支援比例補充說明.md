# 車資與 AI 支援比例補充說明

## 一、車資計算基礎

| 項目      | 金額          | 說明                          |
| --------- | ------------- | ----------------------------- |
| 油資/車資 | 20,000 元     | 每日實地拜訪 2 位，月度 60 次 |
| 停車費    | 3,000 元      | 每次 50 元 × 60 次            |
| 車輛維護  | 5,000 元      | 保養、保險分攤                |
| 其他交通  | 2,000 元      | 大眾運輸、計程車              |

| 項目      | 金額          | 說明                         |
| --------- | ------------- | ---------------------------- |
| 油資/車資 | 8,000 元      | 每日實地拜訪減少，月度 60 次 |

| **總計**  | **15,000 元** | **節省 50%**                 |

## 二、AI 支援比例說明

### 服務模式分配

```
總服務量: 150位長者/人/月

實地拜訪: 60位 (40%)
├─ 高需求長者: 20位 (每週1次)
├─ 中需求長者: 25位 (每兩週1次)
└─ 新案評估: 15位 (首次必訪)

AI語音關懷: 90位 (60%)
├─ 定期問候: 50位 (每週2-3次)
├─ 用藥提醒: 25位 (每日1次)
└─ 健康追蹤: 15位 (每週1次)
```

| 健康追蹤 | 20%  | 監測健康狀況變化   |
| 緊急通知 | 10%  | 重要訊息即時傳達   |
| 娛樂互動 | 10%  | 提升生活品質       |

## 三、車資節省原理

### 路線優化效益

**導入前**: 每日拜訪 2 位，分散各區，平均單程 30 分鐘

- 每日通勤時間: 約 4 小時
- 每月行駛里程: 約 1,200 公里

**導入後**: 每日拜訪 5 位，集中路線規劃，平均單程 15 分鐘

- 每日通勤時間: 約 2 小時（減少 50%）
- 每月行駛里程: 約 600 公里（減少 50%）

### 車資節省計算

```
油資節省 = (1,200km - 600km) × 10元/km = 6,000元
維護節省 = 里程減少50% × 5,000元 = 2,500元
其他節省 = 時間效率提升 = 1,500元
總節省 = 10,000元 + 維護2,000元 + 其他3,000元 = 15,000元
```

### 服務切換機制

1. **AI → 實地**: 當 AI 發現異常時，立即安排實地拜訪
2. **實地 → AI**: 穩定個案可增加 AI 比例，減少實地頻率
3. **動態調整**: 根據長者狀況每月檢討調整
