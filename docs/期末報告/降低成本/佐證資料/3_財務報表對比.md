### 投資回報計算

```
第一年：
總投資 = 建置500,000 + 維護500,000 = 1,000,000元
年度節省 = 2,700,000元
淨收益 = 2,700,000 - 1,000,000 = 1,700,000元
ROI = (1,700,000 / 1,000,000) × 100% = 170%
回本期 = 1,000,000 / 225,000 = 4.4個月

第二年起：
年度成本 = 500,000元（僅維護）
年度節省 = 2,700,000元
淨收益 = 2,200,000元
ROI = (2,200,000 / 500,000) × 100% = 440%
```
