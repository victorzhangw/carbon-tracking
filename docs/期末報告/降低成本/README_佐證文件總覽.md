## 核心數據摘要

### 基本設定

- **社工人數**: 15 人
- **每日工資**: 1,000 元/人
- **導入前車資**: 1,000 元/人/日（月度 30,000 元）
- **導入後車資**: 500 元/人/日（月度 15,000 元）

### 關鍵成效

| 指標       | 導入前      | 導入後    | 改善  |
| ---------- | ----------- | --------- | ----- |
| 每日拜訪量 | 2 位        | 5 位      | +150% |
| 月度拜訪量 | 60 位       | 150 位    | +150% |
| 月度成本   | 60,000 元   | 45,000 元 | -25%  |
| 單位成本   | 1,000 元/人 | 300 元/人 | -70%  |

### 年度效益（15 人團隊）

- **年度節省**: 270 萬元
- **服務增量**: +16,200 人次
- **投資回報率**: 440%
