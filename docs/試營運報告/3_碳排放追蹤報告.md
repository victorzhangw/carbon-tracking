# 3. 碳排放追蹤系統試營運報告

## 3.1 系統功能與特色

### 核心功能

1. **訪視記錄管理**

   - 訪視日期、時間記錄
   - 訪視對象管理
   - 訪視類型分類（實體/語音）

2. **交通工具追蹤**

   - 多種交通工具選擇
   - 里程數記錄
   - 自動碳排放計算

3. **碳排放計算**

   - 依環保署排放係數計算
   - 即時統計與分析
   - 趨勢圖表呈現

4. **統計報表**
   - 每日/每週/每月統計
   - 碳排放趨勢分析
   - 成本效益分析

### 技術特色

- 📊 **即時統計**: 自動計算碳排放量
- 🌍 **環保署標準**: 採用官方排放係數
- 📱 **行動化**: 支援手機 APP 記錄
- 📈 **視覺化**: 圖表化呈現數據

## 3.2 碳排放計算方法

### 排放係數標準

依據「行政院環境保護署溫室氣體排放係數管理表 6.0.4 版」

| 交通工具    | 排放係數 | 單位       | 資料來源 |
| ----------- | -------- | ---------- | -------- |
| 機車        | 0.063    | kg CO₂e/km | 環保署   |
| 汽車        | 0.2171   | kg CO₂e/km | 環保署   |
| 公車        | 0.0573   | kg CO₂e/km | 環保署   |
| 捷運        | 0.0144   | kg CO₂e/km | 環保署   |
| 火車        | 0.0251   | kg CO₂e/km | 環保署   |
| 步行/腳踏車 | 0        | kg CO₂e/km | 零排放   |

### 計算公式

```
碳排放量 (kg CO₂e) = 里程數 (km) × 排放係數 (kg CO₂e/km)
```

### 語音關懷碳排放

- **語音關懷**: 0 kg CO₂e（無交通移動）
- **實體訪視**: 依實際交通工具計算
