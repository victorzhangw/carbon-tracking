## 📊 數據來源

### 從系統後台取得

1. 登入系統管理後台
2. 進入「統計報表」功能
3. 選擇試營運期間
4. 匯出統計數據

### 從資料庫查詢

```sql
-- 訪視記錄統計
SELECT COUNT(*) as total_visits FROM visit_records
WHERE date BETWEEN '開始日期' AND '結束日期';

-- 碳排放統計
SELECT SUM(carbon_emission) as total_carbon FROM visit_records
WHERE date BETWEEN '開始日期' AND '結束日期';

-- 交通工具統計
SELECT transport_type, COUNT(*) as count,
       SUM(distance) as total_distance,
       SUM(carbon_emission) as total_carbon
FROM visit_records
WHERE date BETWEEN '開始日期' AND '結束日期'
GROUP BY transport_type;
```

### 從問卷調查取得

1. 整理長者滿意度調查問卷
2. 整理社工訪談記錄
3. 計算各項指標的平均值與百分比

## 💡 填寫技巧

1. **優先填寫關鍵數據**: 先填寫最重要的 KPI 數據
2. **保持數據一致性**: 確保相關數據在不同章節中一致
3. **提供具體案例**: 在回饋部分提供具體的使用者意見
4. **計算驗證**: 使用試算表驗證計算結果
5. **定期備份**: 填寫過程中定期儲存備份
