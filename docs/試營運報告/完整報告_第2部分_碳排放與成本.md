# 第三章 碳排放追蹤系統試營運報告

## 3.1 系統功能與特色

### 核心功能

1. 訪視記錄管理

   - 訪視日期、時間記錄
   - 訪視對象資料管理
   - 訪視類型分類（實體/語音）
   - 訪視內容摘要記錄

2. 交通工具追蹤

   - 多種交通工具選擇（機車、汽車、公車、捷運、步行）
   - 里程數精確記錄
   - 自動碳排放計算
   - GPS 定位輔助（選用）

3. 碳排放計算

   - 依環保署排放係數計算
   - 即時統計與累計
   - 趨勢圖表呈現
   - 月度/季度/年度報表

4. 統計報表
   - 每日/每週/每月統計
   - 碳排放趨勢分析
   - 交通工具使用分析
   - 成本效益對比

## 3.2 碳排放計算方法

### 排放係數標準

依據「行政院環境保護署溫室氣體排放係數管理表 6.0.4 版」

| 交通工具    | 排放係數 | 單位       | 資料來源 |
| ----------- | -------- | ---------- | -------- |
| 機車        | 0.063    | kg CO₂e/km | 環保署   |
| 汽車        | 0.2171   | kg CO₂e/km | 環保署   |
| 公車        | 0.0573   | kg CO₂e/km | 環保署   |
| 捷運        | 0.0144   | kg CO₂e/km | 環保署   |
| 火車        | 0.0251   | kg CO₂e/km | 環保署   |
| 步行/腳踏車 | 0        | kg CO₂e/km | 零排放   |

### 計算公式

```
碳排放量 (kg CO₂e) = 里程數 (km) × 排放係數 (kg CO₂e/km)
```

### 語音關懷碳排放

- 語音關懷: 0 kg CO₂e（無交通移動）
- 實體訪視: 依實際交通工具計算

## 3.5 交通工具使用分析

### 使用偏好分析

實體訪視主要交通工具:

1. 機車: 65% - 機動性高，適合都會區短程訪視
2. 汽車: 30% - 適合偏鄉長程訪視或多點訪視
3. 公車: 5% - 環保但時間較不彈性

### 區域差異分析

| 區域     | 主要交通工具 | 平均里程  | 月碳排放 |
| -------- | ------------ | --------- | -------- |
| 都會區   | 機車（70%）  | 0.6 km/次 | 85 kg    |
| 偏鄉地區 | 汽車（60%）  | 1.2 km/次 | 44 kg    |

### 優化建議

1. 鼓勵使用低碳交通工具: 增加公車、捷運使用誘因
2. 優化訪視路線: 同區域多位長者可規劃順路訪視
3. 提高語音關懷比例: 非緊急狀況優先使用語音關懷
