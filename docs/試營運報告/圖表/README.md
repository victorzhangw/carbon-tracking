- `長者參與率.png` - 長者參與率柱狀圖

### 3. 成本效益圖表

- `成本對比.png` - 月度成本對比柱狀圖

- `社工滿意度.png` - 社工滿意度橫條圖

### 5. 系統使用圖表

- `使用時段熱力圖.png` - 每日使用時段分布
- `功能使用排行.png` - 功能使用排行圖
- `系統穩定性.png` - 系統穩定性折線圖

## 🔧 圖表生成

使用 `generate_charts.py` 腳本生成所有圖表：

```bash
python generate_charts.py
```

## 📝 圖表規格

- **格式**: PNG
- **解析度**: 300 DPI
- **尺寸**: 1920x1080 或 1200x800
- **字體**: Microsoft JhengHei（微軟正黑體）
- **色彩**: 使用一致的配色方案

## 🎨 配色方案

- 主色: #4A90E2 (藍色)
- 輔色: #51CF66 (綠色)
- 警示: #FF6B6B (紅色)
- 中性: #868E96 (灰色)
