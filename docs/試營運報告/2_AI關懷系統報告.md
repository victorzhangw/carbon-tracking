# 2. AI 關懷系統試營運報告

## 2.1 系統功能與特色

### 核心功能

1. **定期語音關懷**

   - 自動排程外撥
   - 多語言支援（國語、台語）
   - 自然語音互動

2. **AI 廣播劇系統**

   - 雙語有聲書製作
   - 多角色語音生成
   - 情境式關懷內容

3. **語音辨識與分析**
   - 雙引擎 ASR（Whisper + FunASR）
   - 台語/閩南語辨識
   - 長者語音特徵偵測

### 技術特色

- 🎯 **Qwen TTS**: 高品質語音合成
- 🧠 **DeepSeek AI**: 智能對話生成
- 🎙️ **雙引擎 ASR**: 提升辨識準確度
- 📱 **跨平台**: Web + Android APP
