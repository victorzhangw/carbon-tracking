   - File > Open
   - 選擇 `D:\python\Flask-AICares\android_app`
   - 等待 Gradle 同步

2. **建置 APK**

   - Build > Generate Signed Bundle / APK
   - 選擇 APK
   - Create new keystore（第一次）
   - 建置 Release APK

3. **測試 APK**
