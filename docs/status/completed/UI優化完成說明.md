## 🚀 如何查看

1. **啟動系統**

   ```bash
   start_carbon_tracking.bat
   ```

2. **訪問頁面**

   - 首頁：http://localhost:5000/carbon/
   - 儀表板：http://localhost:5000/carbon/dashboard
   - 訪視記錄：http://localhost:5000/carbon/visit-records
   - 統計報表：http://localhost:5000/carbon/statistics

3. **測試不同螢幕**
   - 按 F12 開啟開發者工具
   - 點擊響應式設計模式
   - 測試不同解析度
