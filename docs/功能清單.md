# 系統功能清單

## 核心功能模組

### ✅ 1. 碳排放追蹤系統（必載）

- 訪視記錄管理（新增/編輯/刪除/查詢）
- 自動碳排放計算
- 統計報表（月度/期間/分布）
- 資料匯出（Excel/CSV）
- AI 關懷記錄追蹤
- 社工資料管理

### ⚙️ 2. ASR 語音識別系統

- 雙引擎識別（Whisper + FunASR）
- 閩南語優化識別
- 長者語音特徵檢測
- 批次檔案處理
- 語言自動檢測

### ⚙️ 3. 語音克隆系統

- 語音樣本上傳與處理
- 自動降噪與特徵提取
- 文字轉語音生成
- 多輪對話支援
- 語音模型管理
- 情緒語調控制

### ⚙️ 4. 智慧語音關懷系統

- 關懷排程管理
- 自動化語音關懷發送
- 天氣資訊整合
- 地理位置記錄
- 關懷效果追蹤

### ⚙️ 5. AI 廣播劇系統

- EPUB 電子書上傳
- 自動章節解析
- 多角色語音合成
- 章節音頻生成
- 廣播劇管理

### ⚙️ 6. 語音測試訓練模組

專為語音技術開發和測試設計的工具集，供開發人員和技術人員使用。

**語音分析**:

- 基礎情緒識別（librosa）
- 進階情緒識別（Wav2Vec2）
- 批次情緒分析
- 即時上傳分析
- 情緒信心度評分

**聲音模擬**:

- 文字轉語音（GPT-SoVITS）
- 參考音頻選擇
- 語速控制
- 音頻檔案管理

**完整文檔**: [語音測試訓練模組完整說明](docs/語音測試訓練模組_完整說明.md)

### ⚙️ 8. 認證授權系統

- JWT Token 認證
- 角色型權限控制（RBAC）
- 用戶管理
- 密碼加密
- 預設角色：admin/manager/staff/viewer

### ⚙️ 9. 客服專員管理

- 專員資料管理
- 專員代號管理
- 語音樣本綁定
- 音頻記錄追蹤

### ⚙️ 10. 音頻處理系統

- 多格式音頻上傳
- 自動格式轉換
- 降噪處理
- 音頻資訊管理

### ⚙️ 11. 語音對話系統

- 多輪對話
- 上下文記憶
- 智慧回應
- 對話歷史

## 技術架構

### 後端技術

- Flask (Python Web 框架)
- SQLite (資料庫)
- JWT (認證)
- Flask-CORS (跨域)

### AI 引擎

- GPT-SoVITS (語音合成)
- Whisper (語音識別)
- FunASR (閩南語識別)
- Wav2Vec2 (情緒識別)
- GPT (自然語言處理)

### 前端技術

- HTML5 / CSS3 / JavaScript
- PWA (漸進式網頁應用)
- 響應式設計
- Chart.js (圖表)

## 資料庫結構

### 用戶與權限

- users (用戶)
- roles (角色)
- permissions (權限)
- user_roles (用戶角色關聯)
- role_permissions (角色權限關聯)

### 客服管理

- staff (客服專員)
- audio (音頻記錄)
- voice_models (語音模型)
- voice_generations (語音生成記錄)

### 碳排放追蹤

- visit_records (訪視記錄)
- ai_care_records (AI 關懷記錄)

### 語音關懷

- voice_care_schedules (關懷排程)
- voice_care_records (關懷記錄)

## 部署方式

### 本地開發

```bash
python app.py
```

### Docker 部署

```bash
docker build -t ai-voice-system .
docker run -p 5000:5000 ai-voice-system
```

### 生產環境

- Gunicorn + Nginx
- SSL 憑證
- 負載平衡

---

## 應用場景

1. **社福機構**: 長者關懷、訪視管理、碳排放追蹤
2. **客服中心**: 語音客服、情緒分析、品質監控
3. **內容創作**: AI 廣播劇、有聲書、多角色配音
