### 3. 計算方法說明頁

**需要截取的內容**：

```
┌────────────────────────────────────────────────┐
│ 排放量計算公式                                  │
├────────────────────────────────────────────────┤
│                                                │
│ 溫室氣體排放量 = 活動數據 × 排放係數            │
│                                                │
│ 其中：                                         │
│ • 活動數據：交通工具行駛里程（公里）            │
│ • 排放係數：每公里二氧化碳當量排放量            │
│   (kg CO2e/km)                                │
│                                                │
│ 範例計算：                                     │
│ 機車行駛 100 公里                              │
│ = 100 km × 0.0695 kg CO2e/km                 │
│ = 6.95 kg CO2e                               │
│                                                │
└────────────────────────────────────────────────┘
```

## 截圖操作步驟

### 步驟 1：下載官方文件

1. 前往環保署溫室氣體登錄平台
2. 點選「排放係數」選單
3. 下載「排放係數管理表 6.0.4 版」PDF 檔案

### 步驟 2：開啟 PDF 並截圖

1. 使用 Adobe Reader 或其他 PDF 閱讀器開啟
2. 調整頁面顯示比例至 100% 或 125%（確保清晰）
3. 使用截圖工具（Windows: Win+Shift+S）截取

### 步驟 3：標註重點

1. 使用圖片編輯軟體（如小畫家、PowerPoint）
2. 在使用的數據旁加上紅色框線或箭頭
3. 標註「本報告使用此數據」

### 步驟 4：儲存檔案

```
檔案命名範例：
```
