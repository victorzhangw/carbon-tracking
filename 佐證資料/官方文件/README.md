## 🎨 如何標註

### 使用工具

- Windows 小畫家
- PowerPoint
- Photoshop / Photopea（線上免費）

### 標註方式

1. 開啟截圖
2. 使用紅色框線圈起使用的數據
3. 加上箭頭指向該數據
4. 標註文字「本報告使用此數據」
5. 儲存為 PNG

### 標註範例

```
┌─────────────────────────────┐
│ 移動源排放係數 - 機車        │
├──────────┬─────────────────┤
│ 125cc    │ 0.0695 kg/km   │ ← [紅框] 本報告使用
│ 250cc    │ 0.0890 kg/km   │
└──────────┴─────────────────┘
```
