# 🚀 快速操作指引 - 3 步驟完成佐證資料準備

## ⏱️ 預計完成時間：30-60 分鐘

---

## 第 1 步：系統截圖（15-20 分鐘）

### 📍 操作位置

AI 關懷系統後台

### ✅ 需要截取的畫面（共 4 張）

#### 截圖 1：GPS 追蹤系統

```
路徑：系統後台 > 訪視管理 > GPS追蹤
重點：顯示社工訪視路線、里程數、交通工具
檔名：GPS追蹤_訪視路線.png
```

#### 截圖 2：訪視頻率統計

```
路徑：系統後台 > 統計報表 > 訪視頻率分析
重點：導入前4次/月 → 導入後2次/月
檔名：訪視統計_頻率對比.png
```

#### 截圖 3：碳排放儀表板

```
路徑：系統後台 > 環境效益 > 碳排放統計
重點：總減少里程、碳排放量、趨勢圖
檔名：碳排放儀表板_總覽.png
```

#### 截圖 4：月度報表

```
路徑：系統後台 > 報表中心 > 月度環境效益報表
重點：任選一個月的詳細數據
檔名：月度報表_202403.png
```

### 🔒 注意事項

- 遮蔽長者姓名、詳細地址、電話
- 保留統計數據、日期、里程
- 使用 Win+Shift+S 截圖
- 儲存為 PNG 格式

### 💾 儲存位置

```
佐證資料/系統截圖/
```

## 第 2 步：環保署文件截圖（10-15 分鐘）

### 📍 操作位置

環保署溫室氣體登錄平台

### ✅ 操作步驟

#### Step 1：下載官方文件

1. 開啟瀏覽器，前往：https://ghgregistry.moenv.gov.tw/
2. 點選「排放係數」選單
3. 下載「排放係數管理表 6.0.4 版」PDF

#### Step 2：截取關鍵頁面（共 3 張）

**截圖 1：機車排放係數**

```
頁面：移動源排放係數 - 機車
重點：125cc = 0.0695 kg CO2e/km
標註：用紅框圈起來，加箭頭標註「本報告使用」
檔名：環保署_機車排放係數.png
```

**截圖 2：汽車排放係數**

```
頁面：移動源排放係數 - 小客車
重點：1600cc = 0.1850 kg CO2e/km
標註：用紅框圈起來
檔名：環保署_汽車排放係數.png
```

**截圖 3：大眾運輸排放係數**

```
頁面：移動源排放係數 - 大眾運輸
重點：平均 = 0.0295 kg CO2e/km
標註：用紅框圈起來
檔名：環保署_大眾運輸排放係數.png
```

### 🎨 標註工具

- Windows：小畫家、PowerPoint
- 使用紅色框線或箭頭
- 字體大小：14-16pt

### 💾 儲存位置

```
佐證資料/官方文件/
```

## 🆘 常見問題

### Q1：系統後台找不到 GPS 追蹤功能？

**A**：可以使用以下替代方案：

- 系統匯出的訪視記錄 Excel
- 社工手寫訪視記錄掃描
- 機構提供的統計報表

### Q2：無法下載環保署 PDF？

**A**：可以：

- 直接從網站截圖線上查詢結果
- 使用本資料夾提供的數據對照表
- 聯繫環保署索取紙本文件

### Q3：截圖包含敏感資訊怎麼辦？

**A**：使用以下工具遮蔽：

- Windows 小畫家：用黑色方塊覆蓋
- PowerPoint：插入形狀遮蔽
- 線上工具：Photopea（免費）

### Q4：時間不夠怎麼辦？

**A**：優先順序：

1. 必做：環保署文件截圖（10 分鐘）
2. 必做：系統碳排放儀表板（5 分鐘）
3. 選做：其他系統截圖
4. 選做：機構報表
