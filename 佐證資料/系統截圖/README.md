# 系統截圖資料夾

## 📸 請將以下截圖放入此資料夾

### 必要截圖（至少 4 張）

1. **GPS 追蹤\_訪視路線.png**

   - 顯示社工訪視的 GPS 路線
   - 包含里程數、交通工具類型

2. **訪視統計\_頻率對比.png**

   - 顯示導入前後訪視頻率對比
   - 導入前：4 次/月 → 導入後：2 次/月

3. **碳排放儀表板\_總覽.png**

   - 顯示總碳排放減少量
   - 包含趨勢圖和關鍵指標

4. **月度報表\_202403.png**（或其他月份）
   - 顯示單月詳細數據
   - 包含服務人數、訪視次數、碳減量

### 選用截圖（加分項）

5. 社工個人訪視記錄
6. 區域別統計圖表
7. 交通工具分布圓餅圖
8. 年度趨勢折線圖

---

## 📝 截圖注意事項

### ✅ 必須包含

- 系統標題列
- 日期時間戳記
- 統計數據
- 圖表（如有）

### ❌ 必須遮蔽

- 長者姓名
- 詳細地址
- 電話號碼
- 身分證字號

### 💡 品質要求

- 解析度：至少 1920x1080
- 格式：PNG 或 JPG
- 檔案大小：500KB - 2MB
- 清晰度：文字清晰可讀

---

## 🔍 如何截圖

### Windows

1. 按 `Win + Shift + S`
2. 選擇截圖區域
3. 自動複製到剪貼簿
4. 開啟小畫家貼上
5. 儲存為 PNG

### 替代方法

- 使用 Snipping Tool
- 使用 ShareX（免費軟體）
- 瀏覽器擴充功能
