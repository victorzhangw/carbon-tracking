# AI 關懷系統後台截圖指引

## 需要截取的系統畫面清單

### 1. GPS 追蹤系統介面

**截圖位置**：系統後台 > 訪視管理 > GPS 追蹤

**必須包含的資訊**：

- 社工姓名/編號（可遮蔽敏感資訊）
- 訪視日期與時間
- 出發地點與目的地
- 實際行駛路線（地圖顯示）
- 總里程數（公里）
- 交通工具類型（機車/汽車/大眾運輸）

**截圖範例說明**：

```
┌─────────────────────────────────────────┐
│ GPS訪視追蹤系統                          │
├─────────────────────────────────────────┤
│ 社工編號：SW001                          │
│ 訪視日期：2024/03/15 14:30             │
│ 出發地：台北市中正區○○路               │
│ 目的地：台北市大安區○○街               │
│ 交通工具：機車                          │
│ 行駛里程：7.2 公里                      │
│ [地圖顯示路線]                          │
└─────────────────────────────────────────┘
```

### 2. 訪視記錄統計後台

**截圖位置**：系統後台 > 統計報表 > 訪視頻率分析

**必須包含的資訊**：

- 統計期間（2024 年 1 月-6 月）
- 導入前訪視頻率（每月 4 次）
- 導入後訪視頻率（每月 2 次）
- 服務人數（3,300 人）
- 總訪視次數統計
- 減少次數與比例

### 4. 月度報表範例

**截圖位置**：系統後台 > 報表中心 > 月度環境效益報表

**必須包含的資訊**：

- 報表月份
- 當月服務人數
- 當月訪視次數
- 當月減少里程
- 當月碳減量
- 累計碳減量

## 截圖注意事項

### 資訊保護

- ✓ 可保留：統計數據、日期、里程、碳排放量
- ✗ 需遮蔽：長者姓名、詳細地址、社工個人資訊、電話號碼

### 截圖品質要求

- 解析度：至少 1920x1080
- 格式：PNG 或 JPG
- 清晰度：文字清晰可讀
- 完整性：包含系統標題列和時間戳記

### 檔案命名規則

```
截圖類型_日期_說明.png

範例：
- GPS追蹤_20240315_訪視路線.png
- 訪視統計_202401-06_頻率對比.png
- 碳排放儀表板_20240630_總覽.png
- 月度報表_202403_環境效益.png
```

## 截圖時機建議

1. **即時截圖**：在系統正常運作時立即截取
2. **歷史資料**：從系統後台調閱歷史記錄截圖
3. **報表匯出**：使用系統內建的報表匯出功能
4. **多角度驗證**：同一數據從不同報表截圖交叉驗證

## 替代方案

如果系統目前無法提供完整截圖，可以：

1. 使用系統匯出的 Excel 報表
2. 提供系統操作手冊相關頁面
3. 由系統廠商提供功能說明文件
4. 製作系統功能展示影片（截取關鍵畫面）
