## 🎨 如何添加自訂圖示（可選）

如果你想要使用自訂的綠色圖示，可以按照以下步驟：

### 方法 A：使用 Android Studio（推薦）

#### Step 1: 開啟 Image Asset 工具

```
1. 在 Android Studio 中
2. 右鍵點擊 res 資料夾
3. New > Image Asset
```

#### Step 2: 設定圖示

```
Icon Type: Launcher Icons (Adaptive and Legacy)
Name: ic_launcher

Foreground Layer:
- Source Asset: Image
- Path: 點擊資料夾圖示
- 選擇：D:\python\Flask-AICares\static\icons\icon-512x512.png

Background Layer:
- Source Asset: Color
- Color: #689F38 (綠色)

Options:
- Shape: Circle 或 Square
- Resize: 100%
```

#### Step 3: 完成

```
1. 點擊 Next
2. 點擊 Finish
3. Android Studio 會自動生成所有尺寸的圖示
```

#### Step 4: 更新 AndroidManifest.xml

```xml
<!-- 改回使用自訂圖示 -->
<application
    android:icon="@mipmap/ic_launcher"
    android:roundIcon="@mipmap/ic_launcher_round"
    ...>
```

### 方法 B：線上工具生成

#### Step 1: 使用線上工具

訪問以下任一網站：

- https://romannurik.github.io/AndroidAssetStudio/
- https://icon.kitchen/
- https://easyappicon.com/

#### Step 2: 上傳圖示

```
1. 上傳：static/icons/icon-512x512.png
2. 選擇樣式和顏色
3. 下載生成的 mipmap 資料夾
```

#### Step 3: 複製到專案

```
1. 解壓下載的檔案
2. 複製所有 mipmap-* 資料夾到：
   android_app/app/src/main/res/
3. 更新 AndroidManifest.xml
```
