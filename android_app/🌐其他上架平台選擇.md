## 🚀 立即可用方案

### 方案 A：GitHub Releases（推薦）

**5 分鐘完成：**

```
1. 訪問：https://github.com/victorzhangw/carbon-tracking

2. 點擊 "Releases"

3. 點擊 "Create a new release"

4. 填寫：
   Tag: v1.0.0
   Title: 碳排放追蹤系統 v1.0.0
   Description: 初始版本

5. 上傳：app-release.apk

6. 點擊 "Publish release"

7. 完成！分享連結：
   https://github.com/victorzhangw/carbon-tracking/releases
```

### 方案 B：Google Drive

**3 分鐘完成：**

```
1. 上傳 APK 到 Google Drive

2. 右鍵 > 取得連結

3. 設定為「知道連結的任何人」

4. 複製連結

5. 分享給需要的人
```

### 方案 C：在你的網站上

**10 分鐘完成：**

在 `https://carbon-tracking.onrender.com` 加入下載頁面：

```html
<a href="/download/app-release.apk" download> 下載 Android App </a>
```
