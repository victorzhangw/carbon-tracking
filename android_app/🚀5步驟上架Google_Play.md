## 🎯 超簡單 5 步驟

### Step 1: 建置 AAB（10 分鐘）

Google Play 需要 AAB 格式（不是 APK）。

**在 Android Studio 中：**

```
1. Build > Generate Signed Bundle / APK
2. 選擇 "Android App Bundle"（重要！）
3. 選擇你的 keystore
4. 輸入密碼
5. 選擇 release
6. 勾選 V1 和 V2
7. 點擊 Finish
```

**AAB 位置：**

```
android_app/app/release/app-release.aab
```

### Step 3: 註冊開發者（30 分鐘，$25）

```
1. 訪問：https://play.google.com/console/signup

2. 使用 Google 帳號登入

3. 支付 $25 美金（一次性費用）
   - 信用卡或 PayPal

4. 填寫開發者資訊

5. 完成註冊

6. 等待審核（通常幾小時到 1 天）
```

### Step 4: 建立應用程式（30 分鐘）

**在 Google Play Console：**

```
1. 點擊 "建立應用程式"

2. 填寫：
   - 名稱：碳排放追蹤系統
   - 語言：繁體中文 (台灣)
   - 類型：應用程式
   - 免費或付費：免費

3. 商店資訊：
   - 簡短描述：[貼上準備的]
   - 完整描述：[貼上準備的]
   - 圖示：static/icons/icon-512x512.png
   - Feature Graphic：[上傳你製作的]
   - Screenshots：[上傳至少 2 張]

4. 分類：
   - 類別：工具

5. 聯絡資訊：
   - Email：[你的 Email]
   - 網站：https://carbon-tracking.onrender.com

6. 隱私權政策：
   - 網址：https://carbon-tracking.onrender.com/privacy

7. 內容分級：
   - 完成問卷（全部選「否」）

8. 目標對象：
   - 18 歲以上
```

### Step 5: 上傳並提交（10 分鐘）

```
1. 點擊 "發布" > "正式版"

2. 點擊 "建立新版本"

3. 上傳 AAB：
   - 選擇 app-release.aab
   - 等待上傳完成

4. 版本資訊：
   - 版本名稱：1.0.0
   - 新功能：初始版本

5. 點擊 "提交審核"

6. 等待審核（1-3 天）

7. 審核通過後，App 上架！🎉
```

```

### 2. 隱私權政策

```
