## 🚀 完整上架流程

### 階段 1：準備素材（今天，1-2 小時）

#### 1.1 建置 AAB 檔案（必須）

Google Play 現在要求使用 AAB 格式，不接受 APK。

**在 Android Studio 中：**

```
1. Build > Generate Signed Bundle / APK
2. 選擇 "Android App Bundle"（不是 APK）
3. 選擇你的 keystore
4. 輸入密碼
5. 選擇 release
6. 勾選 V1 和 V2
7. 點擊 Finish
8. 等待建置完成
```

**AAB 位置：**

```
D:\python\Flask-AICares\android_app\app\release\app-release.aab
```

### 階段 2：註冊開發者帳號（今天，30 分鐘）

#### 2.1 註冊 Google Play 開發者

**步驟：**

```
1. 訪問：https://play.google.com/console/signup

2. 使用 Google 帳號登入

3. 閱讀並同意開發者協議

4. 支付註冊費用：$25 美金（一次性）
   - 可使用信用卡
   - 可使用 PayPal

5. 填寫開發者資訊：
   - 開發者名稱
   - Email
   - 網站（可選）
   - 電話（可選）

6. 完成註冊
```

**注意事項：**

```
- 註冊費用 $25 是一次性的
- 帳號審核可能需要 1-2 天
- 審核通過後才能上架 App
```

### 階段 3：建立應用程式（明天，1 小時）

#### 3.1 建立新應用程式

**在 Google Play Console：**

```
1. 點擊 "建立應用程式"

2. 填寫基本資訊：

   應用程式名稱：
   碳排放追蹤系統

   預設語言：
   繁體中文 (台灣)

   應用程式類型：
   ☑ 應用程式

   免費或付費：
   ☑ 免費

3. 聲明：
   ☑ 我已閱讀並同意...

4. 點擊 "建立應用程式"
```

#### 3.2 填寫商店資訊

**應用程式詳細資料：**

```
1. 點擊左側 "商店資訊" > "主要商店資訊"

2. 填寫：

   應用程式名稱：
   碳排放追蹤系統

   簡短說明：
   [貼上你準備的簡短描述]

   完整說明：
   [貼上你準備的完整描述]

   應用程式圖示：
   [上傳 512x512 的圖示]
   （可以使用 static/icons/icon-512x512.png）

   Feature Graphic：
   [上傳你製作的 1024x500 圖片]

   Screenshots：
   [上傳至少 2 張截圖]

3. 點擊 "儲存"
```

**分類：**

### 階段 4：上傳 AAB（明天，30 分鐘）

#### 4.1 建立正式版本

```
1. 點擊左側 "發布" > "正式版"

2. 點擊 "建立新版本"

3. 上傳 AAB：
   - 點擊 "上傳"
   - 選擇：app-release.aab
   - 等待上傳完成

4. 版本名稱：
   1.0.0

5. 版本資訊（這個版本的新功能）：

   初始版本功能：
   - 訪視記錄管理
   - 碳排放計算
   - 統計分析
   - Excel 匯出

6. 點擊 "儲存"
```

### 階段 5：內容分級（明天，15 分鐘）

```
1. 點擊 "政策" > "應用程式內容" > "內容分級"

2. 點擊 "開始問卷"

3. 選擇應用程式類別：
   ☑ 工具、生產力、通訊或其他

4. 回答問題（全部選「否」）：

   是否包含暴力內容？ ☐ 否
   是否包含性內容？ ☐ 否
   是否包含不當語言？ ☐ 否
   是否包含恐怖內容？ ☐ 否
   是否包含賭博內容？ ☐ 否
   是否包含毒品內容？ ☐ 否

5. 點擊 "儲存"

6. 點擊 "提交"
```

### 階段 6：目標對象（明天，10 分鐘）

```
1. 點擊 "政策" > "應用程式內容" > "目標對象"

2. 目標年齡層：
   ☑ 18 歲以上

3. 是否針對兒童設計？
   ☐ 否

4. 點擊 "儲存"
```

### 階段 7：提交審核（明天，5 分鐘）

#### 7.1 最終檢查

```
確認以下項目都完成：
☑ AAB 已上傳
☑ 商店資訊已填寫
☑ 截圖已上傳
☑ 隱私權政策已設定
☑ 內容分級已完成
☑ 目標對象已設定
```

#### 7.2 提交審核

```
1. 點擊 "發布" > "正式版"

2. 檢查版本資訊

3. 點擊 "提交審核"

4. 確認提交
```

## ⏱️ 審核時間

```
一般審核時間：1-3 天
快速審核：數小時
慢速審核：最多 7 天
```

## 📧 審核結果

### 審核通過

```
✅ 恭喜！App 已上架

你會收到 Email 通知
App 會出現在 Google Play 商店
用戶可以搜尋並下載
```

### 審核被拒

```
❌ 需要修改

Google 會說明原因
修改後重新提交
通常是：
- 隱私權政策問題
- 內容分級問題
- 截圖或描述問題
```

## 🆘 常見問題

### Q: 可以用 APK 上架嗎？

### Q: 審核會不會不通過？

A: 只要按照指南填寫，通常都會通過。

### Q: 上架後可以改價格嗎？

A: 免費 App 不能改成付費，但可以加入內購。

### Q: 需要公司才能上架嗎？

A: 不需要，個人開發者就可以。
