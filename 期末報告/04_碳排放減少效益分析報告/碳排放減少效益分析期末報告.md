```

```

## 九、附錄

### 9.1 計算公式彙整

#### 9.1.1 核心計算公式

**加權平均排放係數**:

```
排放係數 = Σ(車輛類型排放係數 × 使用比例)
= (0.0695 × 0.65) + (0.1850 × 0.30) + (0.0295 × 0.05)
= 0.1018 kg CO2e/km
```

**單次訪視碳排放**:

```
單次排放 = 平均距離 × 排放係數
= 15 km × 0.1018 kg CO2e/km
= 1.527 kg CO2e
```

**總碳排放減少**:

```
總減少 = 減少次數 × 單次排放 - 系統排放
= 39,600 次 × 0.001527 噸 - 25.45 噸
= 35.02 噸 CO2e（保守估計後 = 30.23 噸）
```

#### 9.1.2 延伸計算公式

**等效植樹數**:

```
植樹數 = 碳減量 ÷ 單棵樹年吸收量
= 30,230 kg ÷ 22 kg/棵
= 1,374 棵
```

**節約汽油量**:

```
汽油量 = 總里程 ÷ 平均油耗
= 594,000 km ÷ 31.16 km/L
= 19,064 L
```

### 9.3 名詞解釋

**CO2e（二氧化碳當量）**:
將各種溫室氣體的效應統一換算為二氧化碳的當量，便於比較和計算。

**範疇一排放**:
組織直接擁有或控制的排放源所產生的直接溫室氣體排放。

**範疇二排放**:
組織使用外購電力、熱或蒸汽所產生的間接溫室氣體排放。

**範疇三排放**:
組織活動產生的其他間接溫室氣體排放（上下游活動）。

**加權平均**:
考慮各項目比例的平均值計算方法。

**保守估計**:
採用較低的估算值，避免高估效益。
