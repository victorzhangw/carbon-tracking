## 📝 使用說明

### 閱讀順序建議

1. **先讀執行摘要**: 每份報告的 `執行摘要.md` 提供快速概覽
2. **深入主報告**: 閱讀完整的期末報告了解技術細節
3. **查看圖表**: charts/ 目錄包含所有視覺化圖表
4. **驗證數據**: 查看 LOG 檔案確認數據真實性

### 生成圖表

每份報告都包含圖表生成程式：

```bash
# 進入報告目錄
cd 期末報告/01_優化後模型成效比較報告

# 執行圖表生成
python generate_charts_simple.py

# 或使用完整版 (需安裝 seaborn)
python generate_charts.py
```
