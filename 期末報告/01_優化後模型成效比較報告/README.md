## 🚀 快速開始

### 環境需求

```bash
Python 3.8+
matplotlib >= 3.5.0
numpy >= 1.21.0
pandas >= 1.3.0
seaborn >= 0.11.0
```

### 安裝依賴

```bash
pip install matplotlib numpy pandas seaborn
```

### 生成圖表

```bash
cd 期末報告
python generate_charts.py
```
